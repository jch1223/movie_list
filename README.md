### Deploy
https://sleepy-gates-e3bd86.netlify.com/

### `npm install`

node_modules을 설치합니다

### `npm start`

[http://localhost:3000](http://localhost:3000) 로 열립니다

### 설명

- movie api를 이용하여 영화 리스트와 상세 정보를 나타내는 페이지 제작
- 리스트는 인피니티 스크롤 형식으로 
- styled-components andt를 사용하여 레이아웃 및 ui 구성
- react hook 사용
- react-router-dom 사용
